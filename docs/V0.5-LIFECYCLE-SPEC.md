# Consite v0.5 — The Project Lifecycle Spine

**The idea:** the app IS the process. Every project moves through six stages, and each
stage's checklist is VERIFIED BY THE APP from real data — never self-reported ticks.
Everyone sees what's done, what's next, and who's on it. Automation at every step.

## Stages

Contract → Setup → Crew → Build → Punch → Closeout

| Stage | Auto-verified checks (live data, not manual ticks) |
|---|---|
| Contract | client linked (CRM) · contract PDF attached · value/dates entered |
| Setup | ≥1 drawing uploaded · phases created (template) · geofence set · FLHA form assigned |
| Crew | foreman assigned · crew ≥ 2 · first-aid attendant on crew (checked against real certs) |
| Build | all phases marked done (each phase completion notifies office + crew) |
| Punch | zero open deficiencies (enforced — stage can't close around them) |
| Closeout | final QC submitted · audit pack generated · project archives as searchable memory |

## Automation requirements (from the owner)

- FLHA: AI auto-fills from site context + voice (✅ exists) — keep improving guesses
- Toolbox talk: dead-simple form, AI-fillable, auto-stored + emailed like all records
- Every completion (phase, form, task) → automatic updates to everyone affected
  (office inbox, crew notices, WhatsApp when enabled)
- Each check failure shows an INSTRUCTION: not "✗ crew" but "✗ No first-aid attendant —
  assign someone with a valid OFA ticket (Users → role chip)"

## Data model

- `projects/{pid}` += `stage`, `clientId?`, `contractPath?`, `contractValue?`
- `projects/{pid}/phases/{id}`: name, order, targetStart/End, status pending|active|done,
  invoiceMilestone, completedAt/By
- Phase templates in code (`PHASE_TEMPLATES`) — house-framing first; trade packs later.
  This is the SaaS shape: same engine, swappable trade templates.

## Surfaces

- Admin Projects list → tap opens **Project Lifecycle** screen: stage stepper,
  live checklist with instructions, phases (create-from-template, advance, complete),
  advance-stage button gated on checks
- Foreman briefing gains phase line ("Phase 3 — roof · 2 days to target")
- Dashboard project cards show stage + blocking item
